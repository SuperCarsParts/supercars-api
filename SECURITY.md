# Security Policy

The security of SuperCars API is important to us.

If you believe you have discovered a security vulnerability, please do not disclose it publicly.

Instead, report it privately to:

api@supercars.parts

Please include:

- description of the issue
- steps to reproduce
- potential impact
- proof of concept (if available)

We will acknowledge your report as quickly as possible and work to resolve confirmed issues.

Thank you for helping keep SuperCars API secure.
